

```bash
# Setup
cd /tmp && rm -rf vt && mkdir vt && cd vt
VRIT=/Users/dev0/sandbox/claude/mygit/target/debug/vrit
$VRIT init
echo 'user.name = You' >> .vrit/config
echo 'user.email = you@test.com' >> .vrit/config
echo "v1" > file.txt
$VRIT add .
$VRIT commit -m "First"
echo "v2" > file.txt
$VRIT add file.txt
$VRIT commit -m "Second"

# --- RESET ---

# 1. Unstage changes (reset with no args)
echo "v3" > file.txt
$VRIT add file.txt
$VRIT status             # file.txt staged (green)
$VRIT reset
$VRIT status             # file.txt unstaged (red), working tree still has "v3"
cat file.txt             # still "v3" — working tree untouched

# 2. Reset to a previous commit
$VRIT log                # note the First commit SHA
$VRIT reset <first-commit-sha>
$VRIT log                # only First commit — Second is gone
cat file.txt             # still "v3" — working tree untouched
$VRIT status             # file.txt shows as modified (index has "v1", disk has "v3")

# 3. Reset warns about unreachable commits
# (you should have seen the warning in step 2)

# --- STASH ---

# 4. Stash dirty changes
echo "dirty work" > file.txt
cat file.txt             # "dirty work"
$VRIT stash
cat file.txt             # back to "v1" (HEAD content)
$VRIT status             # clean

# 5. Pop restores changes
$VRIT stash pop
cat file.txt             # "dirty work" is back

# 6. Multiple stashes stack
echo "change A" > file.txt
$VRIT stash
echo "change B" > file.txt
$VRIT stash
echo "change C" > file.txt
$VRIT stash
$VRIT stash list         # stash@{0}, stash@{1}, stash@{2}

# 7. Pop is LIFO
$VRIT stash pop
cat file.txt             # "change C" (most recent)
$VRIT stash list         # stash@{0}, stash@{1}
$VRIT stash pop
cat file.txt             # "change B"
$VRIT stash pop
cat file.txt             # "change A"
$VRIT stash list         # empty (no output)

# 8. Stash with staged changes
echo "staged change" > file.txt
$VRIT add file.txt
$VRIT status             # staged
$VRIT stash
$VRIT status             # clean
$VRIT stash pop
cat file.txt             # "staged change" restored

# 9. Error cases
$VRIT checkout -- file.txt
$VRIT reset
$VRIT status             # clean
$VRIT stash              # "no local changes to save"
$VRIT stash pop          # "no stash entries"

# --- COMBINED WORKFLOW ---

# 10. Stash before merge
$VRIT branch feature
$VRIT checkout feature
echo "feature" > feature.txt
$VRIT add feature.txt
$VRIT commit -m "Feature"
$VRIT checkout main
echo "wip" > file.txt
$VRIT stash              # save WIP
$VRIT merge feature      # merge clean
$VRIT stash pop           # restore WIP
cat file.txt             # "wip"
ls feature.txt           # exists from merge
```

Key things to validate:
- **Reset leaves working tree alone** — only index changes
- **Stash/pop is a perfect roundtrip** — file content restored exactly
- **Multiple stashes are LIFO** — pop returns most recent first
- **Stash list numbering** — `stash@{0}` is newest
- **Error messages** are clear for clean tree / empty stack
